# FitBloom - Health & Wellness Platform

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![React](https://img.shields.io/badge/React-18-blue.svg)](https://react.dev/)
[![Node.js](https://img.shields.io/badge/Node.js-20-green.svg)](https://nodejs.org/)

**FitBloom** is a modern, full-stack health & wellness platform built with **React**, **Node.js**, **MongoDB**, and **Tailwind CSS**. Users can book consultations, read expert articles, and track their wellness journey. Doctors/Admins can manage appointments, clients, and publish articles.
